HTML and CSS Cheatsheet

HTML Tables

Tag Name Description
<table> Table Creates a table with the tags in the element
<tr> Table Row Creates the table rows
<td> Table Data Creates the cells in the rows
<th> Table Headings Gives titles to rows and columns. Needs a scope attribute to put with the column or the row. eg. scope = "col" or scope = "row"
<tbody> Table Body Should contain all of the tables data including headings
<thead> Table Head When we section off the body we should also section off the head
<tfoot> Table Footer The table footer usually contains sums, differences and other results

HTML Table Attributes

Attribute Name Description & Values
scope = Scope Scope attribute with the value row or col. Row puts the heading for the row. Col puts the heading to a column.
colspan = Column Spanning The colspan attribute takes an integer and that tells the browser that the data needs to be that many columns across
rowspan = Row Spanning The rowspan attribute takes an integers and tells the browser the data needs to span that many rows

Other HTML tags

Tag Name Description
<body> Body Only content between the body tags will be displayed on the webpage
<h1-h6> Headings 1 - 6 Six different headings. Heading 1 is the largest and they become smaller until heading 6
<div> Div Short for division, useful for grouping elements together specifically for styling CSS. Can contain any text or elements.
<p> Paragraph Displays chunks of text
<span> Span Contains short pieces of text
<br> Line Break Spacing in code doesn't affect how it's displayed on the page but the line break element adds a break in the text.
>ul> Unordered Lists Creates an unordered/ bullet point list. All elements have to be in an li element
>ol< Ordered List Creates an ordered/ numbered list. All elements need to be between li element tags
<li> List Each point in the lists must be between this element to be displayed
<img> Image Adds an image into a webpage, self-closing tag and needs an src attribute with the image source to link.
<video> Video Displays videos, needs an src for the link, a height & width attribute for the video size, a controls attribute for basic controls and in between your video tags add "Video not supported" to display if the video doesn't load.
<!DOCTYPE html> Document Type Declaration Tells the browser what type of document to expect, refers to HTML 5
<html> HTML Tag Everything must be within the html tag
<head> Head Gives the browser information about the page itself, contains metadata and not displayed on the webpage
<title> Title The title of the webpage is always in the head. The title will be displayed on the tab.
<a> Anchor Add links by wrapping the text in an anchor element. Use the href attribute with the link source. Use the target attribute to tell how the link should be opened.
<!-- ... --> Comment Characters between the comment marks will be ignored by the browser. This helps others to understand your code and also allows you to experiment with different codes by commenting it out.
<header> Header Semantic tag. Usually contains introductory content or a navigation element.
<nav> Nav Semantic tag. Used for navigation links or contents tables. Used instead of a div.
<main> Main Semantic tag. Holds the dominant content
<footer> Footer Semantic tag. The footer element usually contains; contact info, copyright info, terms of use, site map or reference to the top of the page links.
<article> Article Semantic Tag. Contains content that makes sense when on it's own
<section> Section Semantic Tag. Can be elements like chapters or elements.
<aside> Aside Semantic Tag. Marks additional information. Common uses being; bibliographies, end notes, comments, pull quotes, editorial sidebars and additional information.
<figure> Figure Semantic Tag. What we put around media.
<figcaption> Figcaption Semantic tag to caption the media in the figure element.
<audio> Audio Embeds audio into the webpage. Needs an src, type and controls attribute
<embed/> Embed Use embed to embed an gif and is a self-closing tag. It used to be for all media but no longer practice.

HTML Attributes

Attribute Name Description and Values
id = "" ID Names an individual element and the highest specification
src="" Source Attribute that links either a local file or URL to an image or audio element.
alt="" Alternative description Makes images and videos more accessible by describing the image or video. Also helps SEO and improves site rankings.
width= Width Sets the width of a video element
height= Height Sets the height of a video
controls Controls Includes basic video or audio controls eg play & pause. You can also use autoplay controls to autoplay the content.
href="" href uses a URL to link the element. Used in the anchor element
target="" target Tells a link how to open it. The "_blank" value opens the link in a new tab
type="" Type In an audio element you have to specify the type of audio. The value can be "audio/mp3"
loop Loop The loop attribute plays the media continuously

CSS Selectors

Selector Name Description
* {} Universal Selector Selects all the elements of any type. Mainly used for resetting the default browser or all children of a parent element.
p {} Type Selector Selects a specific html element to style.
.class{} Class Selector Selects everything with the specific class attribute.
#id{} ID Selector A uniquely styled element. Can only be used once.
:focus Focus Pseudo-class. Matches when an element has focus
:visited Visited Pseudo-class. Styles when the link has been visited
:disabled Disabled Pseudo-class. If the element has been deactivated it will be styled this way
:active Active Pseudo-class. When the user presses down on the mouse it will be styled this way.
:hover Hover Pseudo-class. When the users mouse hovers over the element it will be styled this way.

CSS Properties

Property Description
font-family: Refers to the typeface. The font must be installed on the user's computer or downloaded with the site. Can use web safe fonts as they are supported by most browsers. Enclose the typeface in quotation marks.
font-size: changes to the font size happen in px
font-weight: how bold or thin a text is. The values can be bold, thin, normal or a value of 0 - 1000
text-align: Aligns the text in the element that holds it. Values can be left, center, right or justify.
color: Styles the foreground or text color of the element
background-color: Styles the background color
opacity: How transparent an element is and measured as a decimal from 0-1.
background-image: url("...") make the background of the element an image.
width: The width of the content in px
height: The height of the content in px
padding: The space between the content and the edge of the element. In px.
border: the thickness and style of the border around the content area. Width in px, style color
margin: The space between the border and the outside edge of the element. In px.
overflow: If the content extends the screen how do you view it. hidden, scroll, visible.
border-radius: Modify the corners of a border by px or %
box-sizing: Controls the type of box model the browser should use. The default is content-box but could change to border-box.
position: Changes the position of an element. Relative, absolute, fixed, sticky, static(default)
display: Inline has a box wrapped tightly around the element. Block, fills the entire width. Inline-block makes elements have the ability to appear next to each other.
